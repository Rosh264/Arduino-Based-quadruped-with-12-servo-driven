Arduino-powered-quadruped-with-12-servo-driven
Arduino-powered quadruped with 12 servo-driven joints using inverse kinematics for lifelike walking. Features Bluetooth control via a custom web app, ultrasonic obstacle avoidance, and expressive actions like hand wave, dance & head tilt. Includes 3D print files, circuit diagrams, and step-by-step assembly guide 3D-Printed Spider Robot.
